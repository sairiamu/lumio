# VibePlan – AI-Assisted Visual Planning Tool

## What It Is
A desktop-first visual planning and diagramming tool built with React + Vite + Tauri.
Users design system diagrams, app flows, teaching content, and wireframes on a canvas,
then export clean SVG or self-contained HTML files.

## Core Philosophy
- AI-written codebase: every file must be self-explanatory and modular
- No backend complexity: pure frontend + Tauri for native shell
- Beautiful by default: glassmorphism panels + claymorphism shapes
- Fast to use: keyboard shortcuts, snap-to-grid, intuitive tools

## Target Users
- Developers sketching system architecture
- Teachers designing visual lesson content
- Product managers mapping app flows
- Anyone who wants a beautiful, exportable diagram

## Time Budget
Built in ~3 hours of AI-assisted coding sprints.

## Export Targets
- SVG (vector, scalable)
- Self-contained HTML (single file, shareable)
- JSON (save/load project state)
